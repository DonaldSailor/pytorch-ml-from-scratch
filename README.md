# pytorch-ml-from-scratch

This repository focuses on building machine learning and deep learning concepts in PyTorch from the ground up. The goal is to develop a deep, practical understanding of how modern models work by reimplementing key components rather than relying on high-level abstractions.

The project is based on a series of hands-on tasks and building blocks inspired by exercises from TorchCode, which guide the implementation of progressively more complex concepts.

It starts with fundamental components and gradually progresses toward advanced architectures, including transformer-based models such as GPT. Through this process, the aim is to better understand both the theory and implementation details behind state-of-the-art systems.

## 🎯 Objectives

- Reimplement core PyTorch-like functionality from scratch  
- Build a strong understanding of deep learning fundamentals  
- Follow structured tasks inspired by TorchCode exercises  
- Progressively develop more complex models, up to LLM-level architectures  
- Explore how different design choices impact model behavior and performance  

## 🛣️ Roadmap
